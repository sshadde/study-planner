# Этап 7. Мобильный интерфейс

## Цель этапа

Реализовать мобильный клиент для информационной системы планирования учебных заданий студентов: экраны, навигацию, взаимодействие с REST API, обработку состояний и локальное кэширование для оффлайн-режима.
