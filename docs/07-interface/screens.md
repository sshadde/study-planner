# Описание экранов

## Экран входа и регистрации

Файл: `LoginScreen.kt`

Функции:

- вход по email и паролю;
- регистрация нового студента;
- отображение ошибки авторизации;
- состояние загрузки.

## Список заданий

Файл: `AssignmentListScreen.kt`

Функции:

- просмотр заданий;
- поиск по тексту;
- фильтрация по дисциплине, статусу и приоритету;
- отображение признака offline-cache;
- empty state при отсутствии заданий;
- переход к созданию и деталям задания;
- синхронизация pending-операций и обновление списка.

## Детали задания

Файл: `AssignmentDetailsScreen.kt`

Функции:

- просмотр названия, дисциплины, дедлайна, приоритета, статуса и описания;
- изменение статуса на `DONE` или `IN_PROGRESS`;
- просмотр, создание и удаление напоминаний;
- переход к редактированию;
- удаление задания.

## Создание и редактирование задания

Файл: `AssignmentEditScreen.kt`

Функции:

- ввод названия;
- ввод описания;
- выбор дисциплины из списка существующих дисциплин;
- выбор приоритета (низкий, обычный, высокий);
- выбор даты и времени дедлайна через UI-компонент;
- создание нового задания;
- обновление существующего задания.

## Дисциплины

Файл: `CoursesScreen.kt`

Функции:

- просмотр списка дисциплин;
- добавление дисциплины;
- редактирование дисциплины;
- удаление дисциплины;
- ввод преподавателя дисциплины;
- ввод семестра;
- обновление списка;
- отображение данных из локального кэша.

## Профиль

Файл: `ProfileScreen.kt`

Функции:

- отображение ФИО, email и группы студента;
- выход из учетной записи.

## Навигация

Навигация описана в `StudyPlannerApp.kt`.

| Route | Экран |
|---|---|
| `login` | Вход/регистрация |
| `assignments` | Список заданий |
| `assignments/{id}` | Детали задания |
| `assignments/create` | Создание задания |
| `assignments/{id}/edit` | Редактирование задания |
| `courses` | Дисциплины |
| `profile` | Профиль |
