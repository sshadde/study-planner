# Расширенный глоссарий

| Термин | Определение |
|---|---|
| Access token | JWT-токен, используемый мобильным приложением для доступа к защищенным API |
| Архив | Набор заданий, скрытых из активного списка после завершения или потери актуальности |
| Аутентификация | Проверка личности пользователя по email и паролю |
| Авторизация | Проверка прав пользователя на выполнение действия |
| Валидация | Проверка корректности введенных данных до сохранения |
| Дедлайн | Крайний срок выполнения учебного задания |
| Дисциплина | Учебный предмет, к которому относится задание |
| Задание | Учебная работа, которую студент должен выполнить |
| Кэш | Локальная копия данных на мобильном устройстве |
| Локальное хранилище | База данных мобильного приложения для оффлайн-доступа |
| Напоминание | Запланированная запись о приближении срока задания |
| Оффлайн-режим | Режим работы приложения без подключения к серверу |
| Приоритет | Показатель срочности или важности задания |
| Профиль студента | Данные о студенте: ФИО и учебная группа |
| Просроченное задание | Задание с истекшим дедлайном и невыполненным статусом |
| REST API | Интерфейс HTTP-взаимодействия мобильного клиента и сервера |
| Роль пользователя | Набор прав пользователя в системе |
| Серверная часть | Spring Boot приложение, реализующее бизнес-логику и API |
| Синхронизация | Согласование данных мобильного клиента и сервера |
| Статус задания | Состояние выполнения задания: новое, в работе, выполнено, просрочено, архивное |
| Студент | Основной пользователь системы, управляющий учебными заданиями |
| Уведомление | Сообщение для пользователя о важном событии или дедлайне |
| Фильтрация | Отбор заданий по заданным критериям |
| JWT | Формат токена, содержащий сведения о пользователе и срок действия |
| Swagger UI | Веб-интерфейс для просмотра и тестирования REST API |
