Уточнить проектные решения перед реализацией серверного ядра и мобильного клиента: описать взаимодействие компонентов в ключевых сценариях, определить проектные классы, сигнатуры методов и применяемые паттерны.

## Ключевые сценарии реализации

- Регистрация и вход пользователя с выдачей JWT.
- Создание учебного задания с проверкой владельца дисциплины.
- Поиск и фильтрация заданий по статусу, дисциплине, сроку и тексту.
- Загрузка данных в мобильный кэш и работа при отсутствии сети.
