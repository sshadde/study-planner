## Итоговое состояние

Проект реализует мобильную траекторию:

- Android Native клиент на Kotlin + Jetpack Compose;
- Spring Boot backend на Java 17;
- исполняемый backend JAR;
- PostgreSQL;
- JWT-аутентификация;
- REST API и Swagger UI;
- Room offline cache;
- модульные тесты и JaCoCo coverage более 40%;
- статический анализ Checkstyle без нарушений.
